# UI Kit (baseline)

This project uses Tailwind CSS utility-first styling.

## Components (shared/ui)
- Button: variants `primary | secondary | ghost`, disabled state, focus visible ring
- Input (planned): label, helper/error text, disabled, size variants
- Card (planned): header/body/footer slots

## General rules
- Consistent spacing and typography across pages.
- Accessible focus rings and keyboard interaction.
- Prefer composition over large monolithic components.
